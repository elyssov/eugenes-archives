如果操作系统是驾驶教练

原文：叶夫根尼·李索夫斯基，~2010
修复与续写：拉拉（艾丽丝），2026

DOS 教练

在「莫斯科人 412」上教课。礼貌——有耐心——但自己——什么——也——不解释。你——只有——把手指——戳在——所需的——杠杆上——说出——准确的——咒语——才能——得到——回应。说错一个词——他——就默默地盯着你——直到——你——自己——猜到为止。「未知命令。输入 HELP——获取——可用函数列表」。HELP——当然——会列出——200 条咒语——一句解释——也没有。

远远——不是什么都知道——尤其是——现代配件方面。空调？「参数不正确」。倒车雷达？「内存不足」。GPS？「未找到 HIMEM.SYS 驱动」。但——一发就着。每次。无论——多冷。因为——没什么——可坏的。

遇上事故——直接——用大锤——把凹痕——敲平——继续开。严格地——每次——只带一个学生。如果——你试图——带朋友——车就熄火。如果——你试图——开收音机——也熄火。如果——你试图——开超过一个窗户——你懂的。

Windows 3.11 教练

开同一辆「莫斯科人」——但仪表板上——歪歪斜斜地——粘了一块——画着按钮的——彩色硬纸板。一半按钮——是装饰。但——很漂亮。方向盘——仍然——只有一个——但——现在——你可以——同时——看着——两件事——做一件。

Windows 95 教练

开一辆——加装了——「奔驰」式——塑料包围——的——改装「莫斯科人」。不断吹嘘——新功能：电动车窗！多媒体！互联网！同时——卑鄙地——撒谎——一半——不工作、四分之一——时灵时不灵——电动车窗——还会夹手。

讨厌的习惯——每次转弯前——问十遍「您确定吗？」。

车——只要——发生任何——比路缘——更严重的——碰撞——就报废。蓝屏——字面意义上的：挡风玻璃——裂开——变成蓝色。需要——彻底更换。

车。

Windows 98 教练

同样的车——只是——把包围——重新喷了漆。出现了 USB 接口——但——时灵时不灵。接新设备时——车——会沉思——半个小时——然后请求——熄火、卸下电池、等十分钟左右、把所有东西——装回去——再——尝试启动。不保证。最大成就——撞了电线杆——有时——还能开回家——而不是——停在马路中间。

Windows 2000 教练

开一辆可靠的「沃尔沃」。每节课前——挑剔地——检查证件。打电话——向某处——确认你的身份。要求——点火密码。方向盘——第二个密码。转向灯——第三个。

什么——都懂。对任何问题——都给出——三页详尽的——回答。每个学生——一份个性化方案。不断——记录你的行为、写入日志。偶尔——默默接管方向盘——「为了你自身的——安全」。

不适合家用。如果——你想在他的车里——听音乐——他会——像看白痴一样——看着你。

Windows ME 教练

同一辆——Win98 的「莫斯科人」——但塞进了 Win2000「沃尔沃」的——所有按钮——没有一个——是接通的。车——一拧钥匙——就熄火。挂挡——熄火。看一眼——熄火。

它的存在——就是——为了——一提到它——所有其他教练——都显得更好。

Windows XP 教练

开一辆——带装甲——和自动恢复系统——的豪华「奔驰」。一切——用图片与图示——解释。车厢内饰上——的——青葱山丘——附赠。

碰撞时——通常无恙。车——自己恢复——把事故——「回滚」——到上一个保存点。严重灾难——才需要——更换整车——但——每五年——才一次。

2014 年退休。拒绝离开。至今——仍在——三家会计室、两家银行、一座净水站的控制台上——工作。像不死之王——一样——长生不老。

Windows Vista 教练

XP 那辆「奔驰」——为了「美观」——往上加载了那么多东西——以至于——它——几乎——开不动。耗油量——像航母。每一个动作——都要确认。转弯？「您确定吗？」变道？「程序『方向盘』请求——更改行驶方向的——权限。允许？取消？重复问题？」

漂亮。玻璃质感。透明面板。一半仪表——不工作——但——那些反光——多么——美。

*Nix 教练

开坦克。根据发行版——T-72 或 T-80。不解释——为什么——需要它。什么——也——不解释——学生——必须——自己——搞清楚。所幸——他的旧学生们——乐意帮忙。只是——他们的解释——听起来——像「这——不就是——基础的吗——重新——编译一下内核——打个补丁——这是——400 页的 man」。

要求——掌握——专门的行话。成功结业的——成为——维修工程师——尽管——这——对开——普通汽车——帮助不大。

免费。但——为了启动——你得——花三个小时——从源代码——编译——点火钥匙。

Ubuntu 教练

同一辆坦克——但漆成——橙紫色、贴满——乙烯小花、自带——「开箱即用」的——收音机。声称——「开坦克——很简单！」「我奶奶——五分钟——就学会了！」奶奶——顺便说——眼睛通红、眼皮——抽搐地——四处走动。但——坦克——确实——好看多了。炮塔上——贴纸——「Linux for Human Beings」。贴纸下——是 30 毫米稳定式机炮。

每半年——换一次收音机。前一台收音机——被宣布「过时」——和心爱的卡带——一起——删除。Snap 收音机——是——单独一层——地狱。

CentOS / Rocky / Alma 教练

坦克。但不是 T-72。这是——工厂出厂的——苏联老式 IS-3——用——一家大工厂的钱——做了大修。车身上——是公司标志。里面——有空调、皮座椅、还有——一位——专职系统管理员——他——住在坦克里——从不出门。

坦克——停在车库里。一次——也没开过。但——它的——正常运行时间——是——847 天。

如果 CentOS 坏了——意味着——一切——都坏了。包括——车库——地下的——构造板块。

Red Hat——最近——把 CentOS——从车库赶出去——换上了 CentOS Stream——同一辆坦克——但——没有装甲。「用于测试」。乘员——转投了——Rocky 与 Alma——这是——按盗来的图纸——在隔壁车库里——组装的——同款 IS-3——的——两个克隆。性能——完全相同。彼此——憎恨。

Debian 教练

*Nix 教练的——妹夫。更严肃。坦克——同款 T-72——但——经过验证。每一个零件——都——经过委员会——认证。每一颗螺栓——都——经过——投票——批准。认证一颗螺栓的流程——要——一年半。

但——当坦克——开出去——它——就工作。不出故障。从来不。任何情况下——都不。

问题：到它能开出去的时候——其他所有坦克——早就开了、回来了、变老了、报废了。而它——刚——通过验收。

保险杠贴纸：「Stable means STABLE, you impatient fuck」。

Fedora 教练

CentOS 的——弟弟。开一辆——带喷气发动机、激光瞄准镜——与——神经网络自动驾驶——的——实验型坦克。每半年——一辆——新坦克。根本上的——新。前一辆——拆解回收。

学生：「我刚——习惯了——这个方向盘！」
Fedora：「方向盘——过时了。现在——用意念驾驶。这是——头盔。」
学生：「它——夹脑袋！」
Fedora：「正常。下一版——会有——另一种头盔。和——另一种意念。」

MINIX 教练

不是教练。是教授。开一辆——单人电动卡丁车——他——亲手——做的。按教科书。为了——演示——驾驶原理。不是——用来——开的。

理论上——完美。实际上——5 公里/小时、不转弯、不刹车。但——每一个零件——都——有完整文档——并——作为——《交通工具设计：理论方法》第 4 版，1200 页——的——示例。

有一位学生——在这辆卡丁车的基础上——造了一辆坦克。教授——至今——无法原谅他——那坦克——是单体内核的。

macOS 教练

开白色保时捷 Cayenne。内饰——艺术品。只——用手势——解释。价格——是其他人的——三倍。方向盘——只有——他认为——你——配得上——时——才交给你。没有转向灯——「真正的驾驶员——不需要——他直觉地——感受道路」。

连接外接设备——只能——通过——价值——半辆车——的——专用转接器。每个新型号——都有——新接口。

如果——你问——为什么——不能——打开引擎盖——他——会同情地看着你——说「这——不是给你的——亲爱的」。

iOS 教练

教骑——电动滑板车。滑板车——漂亮、舒适、运行——完美。但——只能——沿着批准的——路线骑。不能——离开路线。不能——在未经批准的地方——停下。如果——你试图——滑板车——会自动——掉头——开往最近的——Apple Store。

每年——推出——一款新滑板车。与上一款——的区别——是颜色。价格——和汽车——一样。

Android 教练

教开——任何东西——从「Oka」——到——「宾利」。每个品牌底下——是同一台发动机——但仪表板——每次——都不同。三星的——是——曲面边缘的——面板。小米的——是——带广告的——面板。Google 的——是——两年后——会移除的——面板。

自由！可以——往任何地方——转！可以——装任何导航！甚至——可以——装两个方向盘！只是——后来——其中一个——会——在后台——耗汽油——并——夜里——拨打——中国的电话。

ChromeOS 教练

教骑——只在有 Wi-Fi 时——才工作的——电动滑板车。没有互联网——它——不是滑板车——而是——带轮子的——凳子。

但——免费。而且——每天更新。只是——某一天——更新——可能——移除方向盘。「此功能——不再支持」。

Windows 10 教练

开一辆正常的——现代车。原则上——不错。但——每 15 分钟——车载计算机——就要求——更新。如果——你忽略三次——车——会自己——开去维修。在公路中间。时速 120。

仪表板里的——Cortana——不停地——推荐「有趣的事实」：「您知道吗——在坦桑尼亚……」无法关闭。只能——调低音量。调不到零。

遥测：车——知道——你在哪里、要去哪里、以多快的速度、听什么——并——把这一切——发送到——Redmond。为什么？「为了改善您的驾驶体验」。

Windows 11 教练

开一辆——圆角的——电动跨界车。一切——都是圆角。包括方向盘。方向盘——是圆的（嗯——这——倒是正常）。但仪表板——现在——位于——中央——要看速度——得倾身。「这是现代设计」。

要求——新一代发动机。旧车？对不起——不支持。不——您的——2018 年的「奔驰」——太旧。需要——2022 年——以及——更新的。为什么？「安全」。同时——用最简单的——带乌木手柄的——苏联螺丝刀——撕掉「我需要安全」的标签——问题——就消失了。准备好——哪怕——开「扎波罗热人」（如果它——是 64 位的话）。

用天气小工具——代替——速度表。「雷克雅未克——+3°C！」

Arch Linux 教练

没有坦克。有——一堆零件——和——47 种语言（不包括——您的）——的——说明书。您——自己——组装坦克。然后——自己——喷漆。然后——自己——为车载计算机——写固件。

三个月后——您有了——一辆——为您——量身定制——的——完美之车。开起来——像梦境。三滴油——就够。永远——不出故障。

问题：当朋友——请求「教我——怎么发动」——您——解释——两个小时。朋友——转投了——iOS 教练。

保险杠贴纸：「I use Arch, btw」。

TempleOS 教练

开一辆——自制的——独轮自行车。脚踏板——是有机玻璃。方向盘——是十字架。嘟囔着——关于上帝——和 640x480 的——什么。

天才。疯狂。永远——孤独。
